Tom Wicks

Hi! I am a First Class Graduate of Computer Science, and a hobbyist game developer. I released my first proper game in 2020, but I've been working on various projects as far back as 2017, and I've been coding even further still.
Alongside my studies, I've also essentially taught myself pixel art and music composition, as well as developing an eye for small details and a meticulous approach to problem solving. You can see a selection of my work below.

Previous Games

Cursed Chamber
A platforming prototype done as both a game jam entry, and as a learning exercise in Godot.
Features respawning, wall crawling enemies and flick-screen camera movement.
Desktop users can play it here.

Abyssal Duck Hunt
A (quite tough) boss fight, using PyGame. The game jam it was made for required a duck to be present... so here we are.
Features collision and platformer movement, coded from scratch.

Mimic Masher Maze
Inspired by the classic 3D Monster Maze, this endless dungeon crawler turns the formula on it's head by casting you as the hunter.
Features procedurally generated mazes, a wandering AI, and pseudo-3D graphics.

Doodle Fight
A quick set of turn based battles, with the catch that you draw your character beforehand to define their stats.
Features drawing mechanics coded from scratch.

Lost Property
A short RenPy visual novel about a private eye and a lost wallet. Technically not my first game, but the first one I shipped online.
Features original graphics and a full original soundtrack.


Previous Projects

Neo Robin Hood
A stealth-puzzle game prototype, made in Unity.
Features line-of-sight mechanics and tile-based movement.

Poly Fishtank
A 3D rendering demo coded using OpenGL, C++ and GLSL.
Features coloured lighting, shadow mapping, transparency, and models generated from both OBJ files and procedural generation.

The UoN Archaeology Museum Quiz
A quiz system made in Godot as part of my coursework, designed for a table-mounted touchscreen.
Features a file-based system for adding new questions.

Yie Ar Kung Fu
A loose remake of the classic fighting game, made in C++ as part of my coursework.
Features pixel-perfect hit detection, adaptive enemy AI and character sprites I spent a little too much time on.
